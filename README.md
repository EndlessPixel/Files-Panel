# Files-Panel

跨平台文件管理器，基于Node.js和Express.js开发，提供直观的Web界面用于文件管理操作。

## 功能特点

- 🔍 **文件浏览**：支持文件夹导航、文件列表显示
- 📁 **文件操作**：创建、删除、重命名、复制、剪切、粘贴文件和文件夹
- 📄 **文件编辑**：支持文本和16进制模式编辑
- 📤 **文件上传**：支持文件上传到当前目录
- 📥 **文件下载**：支持单个文件下载
- 🗜️ **压缩解压**：支持ZIP文件的压缩和解压
- 📊 **文件排序**：支持按文件名、大小、创建日期、修改日期、类型排序
- 👁️ **隐藏文件**：支持显示/隐藏隐藏文件
- 📍 **路径存储**：使用localStorage存储访问路径，刷新后不丢失
- 🔗 **路径共享**：支持通过URL参数共享路径

## 技术栈

- **后端**：Node.js, Express.js
- **前端**：HTML, CSS, JavaScript
- **压缩工具**：7-Zip
- **依赖管理**：npm

## 安装说明

1. **克隆仓库**
   ```bash
   git clone https://github.com/EndlessPixel/Files-Panel.git
   cd Files-Panel
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **复制7-Zip文件**
   将7z.exe和7z.dll文件复制到bin目录下
   - Windows: 从 `C:\Program Files\7-Zip` 复制
   - Linux: 安装7-Zip并确保7z命令可用

4. **启动服务**
   ```bash
   npm start
   ```

5. **访问应用**
   打开浏览器访问 http://localhost:3000

## 使用说明

### 基本操作
- **导航**：点击文件夹进入，点击面包屑返回上级目录
- **创建**：点击"新建文件"或"新建文件夹"按钮
- **选择**：点击文件名选择文件，点击复选框进行多选
- **操作**：悬停在文件上显示操作按钮，包括下载、复制、剪切、重命名、删除、编辑等
- **排序**：点击表头进行排序，再次点击切换排序顺序
- **显示隐藏文件**：勾选"显示隐藏文件"复选框

### 高级功能
- **压缩文件**：选择文件或文件夹，点击"压缩"按钮
- **解压文件**：选择ZIP文件，点击"解压"按钮
- **编辑文件**：选择文件，点击"编辑（文本）"或"编辑（16进制）"按钮
- **计算文件夹大小**：点击文件夹的"计算大小"链接

## 注意事项

- **安全**：应用会检测危险目录，禁止在危险目录中进行操作
- **性能**：计算大文件夹大小时可能会耗时较长
- **兼容性**：支持Windows和Linux操作系统
